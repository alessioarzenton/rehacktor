# Rehacktor

A videogames streaming platform built with React, developed during the Aulab course.

## Features

- Browse videogame streams
- Responsive interface
- Dynamic content loading

## Tech Stack

- **React** — UI library
- **Vite** — Build tool
- **PHP** — Backend

## Getting Started

```bash
# Clone the repo
git clone https://github.com/alessioarzenton/rehacktor.git
cd rehacktor

# Install dependencies
npm install

# Start dev server
npm run dev
```
